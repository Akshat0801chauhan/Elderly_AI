# 🧠 Elderly AI Assistant

An AI-powered application designed to assist elderly individuals in their daily lives by providing smart monitoring, reminders, and real-time insights.

---

## 🚀 Features

* 👤 **Face Recognition System**
  Identify visitors using AI-based image recognition.

* 📅 **Daily Activity Tracking**
  Monitor and summarize daily activities of elderly individuals.

* 🚨 **Emergency Detection & Alerts**
  Detect unusual behavior and notify caregivers instantly.

* 💬 **AI Chat Assistant**
  Smart conversational assistant for reminders and support.

* 📍 **Visitor Logging System**
  Automatically logs visitors with timestamp and identity.

* 🔔 **Smart Notifications**
  Medication reminders, alerts, and suggestions.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Spring Boot
* **Database:** PostgreSQL
* **Security:** Spring Security + OAuth
* **AI Integration:** LLM APIs / ML Models
